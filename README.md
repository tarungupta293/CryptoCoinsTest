# CryptoCoins Test

Application fetching top 5 crypto coins from [coingecko]{https://www.coingecko.com/} and showing in application.

Currently, I have used demo account with demo apis to present top 5 real-time cryptocurrencies.

## Installation

    $ git clone https://github.com/tarungupta293/CryptoCoinsTest.git
Download the project from the github and setup in Android Studio.
Install the project .apk in the device and open the application.

## Features Implemented:

1. Clear and detailed Clean Architecture with architectural layers.
2. Retrofit for api service and JSON response.
3. Cache4K to cache the response fetched from api service. Cache valid till 24 hours.
4. ComposeUI for clear and advanced UI presentation.

## UI Components:

1. Image: Presenting crypto coin image
2. Name: Crypto Coin name
3. ATH: All time high value of crypto coin
4. ATH_date: Date on which the crypto coin was at the ATH value.
